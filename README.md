# Deyim Dedektifi: Anlamı Kaybolan Şehir

Ortaokul öğrencileri için deyim ve atasözlerini hikâye tabanlı, oyunlaştırılmış bir yapıyla öğreten Türkçe eğitim uygulaması.

## Özellikler

- Hikâye tabanlı bölüm sistemi
- Deyim ve atasözü soruları
- Doğru/yanlış geri bildirimi
- Puan sistemi
- Rozet koleksiyonu
- localStorage ile ilerleme kaydı
- Mobil uyumlu tasarım

## Kurulum

```bash
npm install
```

## Çalıştırma

```bash
npm run dev
```

## Yayın için build alma

```bash
npm run build
```

## Ön izleme

```bash
npm run preview
```
